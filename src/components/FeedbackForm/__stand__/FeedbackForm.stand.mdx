## Тип виджета

За тип виджета отвечает свойство `type`. Варианты:

- `NPS` — шкала от 0 до 10,
- `CSI` — шкала из 5 звёзд,
- `combo` — обе шкалы: из 5 звёзд и от 0 до 10.

### NPS: шкала от 0 до 10

```tsx
export const FeedbackFormExampleTypeNps = () => {
  const [isOpen, setIsOpen] = useState<boolean>(false);

  return (
    <>
      <FeedbackForm
        label="Как вам наш модуль по созданию сценариев?"
        type="NPS"
        isOpen={isOpen}
        onClose={() => setIsOpen(false)}
      />
      <Button label="Открыть виджет" onClick={() => setIsOpen(true)} />
    </>
  );
};
```

## Содержимое

### Заголовок

Поменять заголовок виджета можно с помощью свойства `label`.

```tsx
export const FeedbackFormExampleLabel = () => {
  const [isOpen, setIsOpen] = useState<boolean>(false);

  return (
    <div>
      <FeedbackForm
        label="Здесь может быть ваш текст"
        isOpen={isOpen}
        onClose={() => setIsOpen(false)}
      />
      <Button label="Открыть виджет" onClick={() => setIsOpen(true)} />
    </div>
  );
};
```

<FeedbackFormExampleLabel />

### Поле для отзыва

Чтобы показать в виджете поле, в котором пользователь может написать отзыв,
добавьте `withOpenQuestion`.

Заголовок перед этим полем можно поменять в свойстве `openQuestionTitle`.

```tsx
export const FeedbackFormExampleQuestion = () => {
  const [isOpen, setIsOpen] = useState<boolean>(false);

  return (
    <div>
      <FeedbackForm
        label="Пример с вопросом"
        isOpen={isOpen}
        withOpenQuestion
        openQuestionTitle="Заголовок вопроса"
        onClose={() => setIsOpen(false)}
      />
      <Button label="Открыть виджет" onClick={() => setIsOpen(true)} />
    </div>
  );
};
```

## Свойства

| Свойство                                 | Тип                                                                                                 | По умолчанию | Описание                                                                    |
| ---------------------------------------- | --------------------------------------------------------------------------------------------------- | ------------ | --------------------------------------------------------------------------- |
| [`title?`](#заголовок)                   | `string`                                                                                            | -            | Заголовок для виджета                                                       |
| [`type?`](#тип-виджета)                  | `'NPS'`, `'CSI'`, `'combo'`                                                                         | `NPS`        | Тип виджета, определяет, что будет внутри: числа, звёзды или и то, и другое |
| [`withOpenQuestion?`](#поле-для-отзыва)  | `boolean`                                                                                           | -            | Добавляет поле для отзыва                                                   |
| [`openQuestionTitle?`](#поле-для-отзыва) | `string`                                                                                            | -            | Заголовок перед полем для отзыва                                            |
| `isOpen?`                                | `boolean`                                                                                           | -            | Открыт ли виджет                                                            |
| `onClose?`                               | `() => void`                                                                                        | -            | Действие при закрытии виджета                                               |
| `onSubmit?`                              | `(args: { e: React.MouseEvent; data: { CSI?: number; question?: string; NPS?: number; }}) => void;` | -            | Действие для отправки результатов заполнения виджета                        |
