# Analytic UI | Дизайн-система Consta

Компоненты для аналитики и обратной связи, входят в [дизайн-систему Consta](https://consta-uikit.vercel.app/).

[Посмотреть на GitHub](https://github.com/consta-design-system/analytic-ui)

## Быстрый старт

#### Установите пакет и зависимости

`yarn add @consta/uikit @consta/analytic-ui` или `npm install @consta/uikit @consta/analytic-ui`

#### Подключите тему

Тему обязательно нужно подключать в корне приложения, так, чтобы все компоненты `@consta/analytic-ui` были вложены в компонент `Theme`.

[Как подключить тему](https://consta-uikit.vercel.app/?path=/docs/thematization-how-to-connect-theme--page)
